# MiniClass

MiniClass é uma plataforma EAD desenvolvida com Django para praticar conceitos de desenvolvimento backend, modelagem de banco de dados e regras de negócio.

## 🚀 Funcionalidades

* Sistema de usuários e perfis
* Perfis de Aluno e Professor
* Cadastro de cursos
* Matrícula por código de acesso
* Listagem de cursos matriculados
* Criação de lições
* Criação de exercícios
* Envio de respostas pelos alunos

## 📊 Modelagem

```text
User
↓
Profile
↓
Enrollment
↓
Course
↓
Lesson
↓
Exercise
↓
Submission
```

## 🛠 Tecnologias

* Python
* Django
* SQLite
